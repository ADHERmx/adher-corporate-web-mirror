== Description ==

ADHER Powder Coating is a professional one-page WordPress theme designed for powder coating and industrial services companies. Features include:

* Responsive design
* Custom hero section
* Services showcase
* Facilities gallery
* Contact information
* Smooth scrolling navigation
* Customizer integration
* Mobile-friendly menu

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the 'Themes' menu in WordPress
3. Copy all images from your React project's src/assets folder to the theme's assets folder
4. Go to Appearance > Customize to configure:
   - Hero section (title, subtitle, background image)
   - Contact information (phones, email, address, WhatsApp)
   - Upload your logo under Site Identity > Logo
5. Go to Appearance > Menus to create your navigation menu

== Required Files ==

After uploading the theme, you need to copy the following image files from your React project to the WordPress theme:

Create an 'assets' folder in the theme directory and copy:
- adher-logo.png
- hero-powder-coating.jpg
- facility-monorail-line.jpg
- facility-curing-oven.jpg
- facility-blasting-room.jpg
- facility-quality-control.jpg
- facility-shipping-area.jpg

Also copy the fonts folder:
- assets/fonts/GlacialIndifference-Regular.otf
- assets/fonts/GlacialIndifference-Bold.otf

== Customization ==

Navigate to Appearance > Customize to modify:

1. Site Identity
   - Site title and tagline
   - Upload your logo

2. Hero Section
   - Hero title
   - Hero subtitle
   - Hero background image

3. Contact Information
   - Phone numbers
   - WhatsApp number
   - Email address
   - Physical address

== Menu Setup ==

1. Go to Appearance > Menus
2. Create a new menu
3. Add Custom Links with these anchors:
   - #inicio (Inicio)
   - #nosotros (Nosotros)
   - #servicios (Servicios)
   - #instalaciones (Instalaciones)
   - #contacto (Contacto)
4. Assign the menu to "Primary Menu" location
